Privacy Policy

Last updated: May 2025

Overview

Web To Prompt is a Chrome extension that helps you capture website elements and generate AI prompts to rebuild them. We are committed to protecting your privacy.

Data We Collect

Web To Prompt does not collect, transmit, or store any personal data on external servers. The extension operates entirely within your browser.

Local Storage Only

When you capture a web element, the captured DOM and style data is stored temporarily in your browser's local storage (chrome.storage.local). This data:

  • Never leaves your device
  • Is never sent to any server
  • Is overwritten each time you capture a new element
  • Is deleted when you uninstall the extension

Permissions

The extension requests the following permissions:

  • activeTab — to read the DOM of the page you are currently viewing, only when you activate capture mode
  • scripting — to inject the capture script into the active tab on demand
  • storage — to temporarily save the captured element data locally in your browser
  • clipboardWrite — to copy the generated prompt to your clipboard when you click "Copy Prompt"

Third-Party Services

Web To Prompt v1.0 does not communicate with any third-party service or API. All prompt generation happens locally in your browser using pure JavaScript.

Contact

If you have questions about this privacy policy, please open an issue on the project's GitHub repository.
